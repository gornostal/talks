Найпростіший спосіб використання


Змінити файли vim / emacs / etc
Додати файли в staging area git add (file)
Переглянути зміни git status / git diff
Створити коміт git commit

Ще простіший спосіб


Змінити файли vim / emacs / etc
Створити коміт git commit -a
